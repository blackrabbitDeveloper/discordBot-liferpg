# Life RPG Discord 서버 운영 가이드

## 1. 서버 생성

**서버 이름**: Life RPG Lab (또는 원하는 이름)

## 2. 채널 구성

```
📋 정보
├── #공지
├── #사용방법
└── #자주-묻는-질문

🎮 플레이
├── #오늘-플레이
└── #자유채팅

📝 피드백
└── #피드백
```

### 채널별 용도

| 채널 | 용도 | 권한 |
|------|------|------|
| #공지 | 업데이트/점검 안내 | 운영자만 쓰기 |
| #사용방법 | 봇 사용법 안내 | 운영자만 쓰기 |
| #자주-묻는-질문 | FAQ | 운영자만 쓰기 |
| #오늘-플레이 | 레벨업 축하 등 봇 메시지 | 봇 + 읽기 전용 |
| #자유채팅 | 유저끼리 대화 | 모두 쓰기 |
| #피드백 | 버그 제보/개선 제안 | 모두 쓰기 |

## 3. 역할(Role) 설정

| 역할 | 색상 | 용도 |
|------|------|------|
| 운영자 | 빨강 | 서버 관리 |
| 모험가 | 초록 | 온보딩 완료 유저 |
| 테스터 | 파랑 | 초기 테스트 참여자 |

## 4. 봇 초대

Discord Developer Portal에서 봇의 OAuth2 URL 생성:
- Scopes: `bot`, `applications.commands`
- Permissions: `Send Messages`, `Embed Links`, `Use Slash Commands`

## 5. 채널 콘텐츠

### #사용방법

```
🎮 Life RPG에 오신 걸 환영합니다!

현실의 행동을 퀘스트로 바꿔 매일 성장하는 봇이에요.

━━━━━━━━━━━━━━━━━━

📌 시작하기
1. 아무 채널에서 /start 입력
2. 5단계 질문에 답하기 (1분이면 끝!)
3. DM으로 첫 퀘스트가 도착해요

━━━━━━━━━━━━━━━━━━

🔄 매일 이렇게 돌아가요
• 아침 8시 — "오늘의 흐름" 선택 DM
• 퀘스트 3개 도착 — 버튼으로 완료/교체/건너뛰기
• 저녁 9시 — 오늘 결과 리포트 DM
• 일요일 — 주간 요약 리포트 DM

━━━━━━━━━━━━━━━━━━

🎯 명령어
• /start — 시작 (또는 처음부터 다시)
• /status — 레벨/XP/스탯 확인
• /goal — 목표 변경
• /pause — 쉬기 모드 (알림 중단)

━━━━━━━━━━━━━━━━━━

💡 팁
• 하루에 퀘스트 1개만 해도 OK!
• 못한 날도 괜찮아요 — 스트릭은 천천히 변해요
• "오늘은 쉬어갈래요" 선택하면 스트릭 보호돼요
• DM이 안 오면 서버 설정에서 DM 허용을 확인하세요
```

### #공지 (첫 글)

```
👋 안녕하세요! Life RPG 테스트에 참여해주셔서 감사합니다.

이 봇은 매일 작은 행동을 퀘스트로 추천하고,
완료하면 경험치와 스탯이 올라가는 자기계발 RPG예요.

📌 참여 방법
1. /start 입력하고 온보딩 완료
2. 매일 DM으로 오는 퀘스트 진행
3. 버그나 의견은 #피드백 에 남겨주세요

현재 MVP 테스트 중이라 부족한 점이 있을 수 있어요.
여러분의 피드백이 봇을 성장시킵니다!
```

### #자주-묻는-질문

```
❓ 자주 묻는 질문

Q: DM이 안 와요
A: Discord 설정 → 개인정보 보호 및 안전
   → "서버 멤버가 보내는 다이렉트 메시지" 활성화

Q: 퀘스트가 마음에 안 들어요
A: 🔄 "다른 걸로" 버튼을 누르면 교체할 수 있어요 (하루 3회)

Q: 오늘은 하기 싫어요
A: 아침 메시지에서 "오늘은 쉬어갈래요"를 선택하세요.
   스트릭이 보호됩니다.

Q: 스트릭이 깨졌어요
A: 하루 못해도 바로 깨지지 않아요.
   1일 → 보호, 2일 → 감소, 3일 → 리셋
   다시 시작하면 금방 회복됩니다!

Q: 목표를 바꾸고 싶어요
A: /goal 명령어로 언제든 변경 가능해요.

Q: 처음부터 다시 하고 싶어요
A: /start를 다시 입력하면 리셋 확인 후 재시작할 수 있어요.
   (레벨/XP/스트릭 모두 초기화됩니다)

Q: 쉬고 싶어요 (며칠간)
A: /pause로 쉬기 모드 전환. 알림이 중단됩니다.
   다시 /pause 하면 복귀해요.
```

## 6. 운영 체크리스트

### 서버 개설 직후

- [ ] 채널 생성 (위 구조대로)
- [ ] 역할 설정
- [ ] 봇 초대
- [ ] #사용방법, #공지, #자주-묻는-질문 작성
- [ ] 본인이 `/start` → 전체 플로우 확인
- [ ] DM 수신 확인

### 유저 초대 시

- [ ] 초대 링크 생성 (만료 없음 권장)
- [ ] 초대 메시지에 "#사용방법 확인 후 /start" 안내
- [ ] 첫 3명은 직접 온보딩 과정 관찰

### 매일

- [ ] #피드백 채널 확인
- [ ] Railway 대시보드에서 봇 상태 확인
- [ ] `/analyze`로 데이터 확인 (유저 5명+ 이후)

### 1주 후

- [ ] `/analyze days:7` 실행
- [ ] JSON 결과를 AI에게 전달하여 분석
- [ ] Day1~3 유지율 확인
- [ ] 피드백 채널 종합

### 2주 후

- [ ] Day7 유지율 확인
- [ ] 가장 많이 교체/건너뛴 퀘스트 확인 → 수정 또는 제거
- [ ] 난이도별 완료율 확인 → 보상 밸런스 조정 여부 판단

## 7. 유저 모객

### 추천 채널

- 주변 지인 5~10명 (초기 피드백)
- 자기계발/생산성 Discord 서버
- 한국 인디게임/사이드프로젝트 커뮤니티
- 트위터/X 개발자 커뮤니티

### 초대 메시지 예시

```
매일 작은 행동을 RPG 퀘스트처럼 추천해주는 Discord 봇을 만들었어요.

• 아침에 오늘 컨디션에 맞는 퀘스트 3개 추천
• 완료하면 XP/레벨/스탯 성장
• 못해도 괜찮아요 — 부드럽게 이어가는 구조

MVP 테스트 참여자를 모집합니다!
1분이면 시작할 수 있어요.

[서버 초대 링크]
```

## 8. 트러블슈팅

| 증상 | 원인 | 해결 |
|------|------|------|
| 봇이 오프라인 | Railway 서비스 중단 | Railway 대시보드 확인, 재배포 |
| /start 응답 없음 | 슬래시 커맨드 미동기화 | 봇 재시작 (자동 sync) |
| DM 안 옴 | 유저 DM 차단 | FAQ 안내 |
| 퀘스트가 안 옴 | paused 상태 | `/pause`로 복귀 |
| 아침 메시지 시간 안 맞음 | 타임존 | scheduler.py KST 설정 확인 |
| DB 에러 | PostgreSQL 연결 | Railway에서 DATABASE_URL 확인 |
